## 📖 Overview

This repository contains the complete source code for the research paper:

> **"Almost Sure Uniform Consistency of Kernel Density Estimation for Outcome Densities Induced by Quantum Measurements"**

We provide a rigorous theoretical proof of the almost sure uniform consistency of the Kernel Density Estimator (KDE) in the context of quantum measurements, along with comprehensive numerical validation on:

- **5 distinct quantum state families**: Schrödinger cat, squeezed, fermionic BCS-like, continuous-variable entangled, and non-Gaussian thermal states.
- **Real experimental data**: Homodyne detection on a Schrödinger cat state (101 samples from Cattiaux et al., 2023).

The results demonstrate that KDE converges uniformly across all cases, outperforms misspecified parametric models (Gaussian MLE, GMM-2, GMM-4), and exhibits remarkable robustness to non-Gaussianity and noise.
